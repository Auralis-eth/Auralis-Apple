# Privacy and Data Collection

This app does not collect, transmit, or share any user or device data. There are no analytics, telemetry, crash reporting, or third‑party tracking SDKs integrated in the application.

What we do not collect:
- No usage analytics or event tracking
- No crash logs sent to third parties
- No identifiers (IDFA, device identifiers)
- No location data
- No contact info or personal data

Networking is limited to fetching user‑initiated content (e.g., audio files and artwork) from the sources the user selects. Caching occurs on-device only to improve playback reliability and performance.

App Store Connect disclosure guidance:
- Data Collection: None
- Data Linked to You: No
- Data Used to Track You: No

If you have questions about privacy, please open an issue in this repository.
